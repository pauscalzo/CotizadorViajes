Se trata de un cotizador de paquetes turísticos para el verano 2024.
Al ingresar el destino deseado se despliegan los paquetes disponibles para dicho destino. Luego se colocan la cantidad de niños y adultos y se calcula el costo del paquete. Estas cotizaciones pueden borrarse con la cruz que tienen al lado.
